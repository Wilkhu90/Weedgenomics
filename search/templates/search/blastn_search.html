Transcripts can be searched via nucleotide search string. Please use 'T' instead of 'U' as your query. Searches are implemented using BLASTn to search a local blast database for each individual species. Local BLAST databases were set up using makeblastdb function from NCBI BLAST+ v. 2.6.0 (NCBI; makeblastdb –in fasta file –parse_seqids –dbtype nucl). If no results are found consider increasing the expected value to allow for less sequence similarity (Default e-value: 0.00001).
